# How to run the repo
1️⃣ Running the Backend
Steps:
Open a terminal.

Navigate to the backend folder:
cd backend
Install dependencies:
npm install
Run the backend server:
npm start

2️⃣ Running the Frontend
Steps:
Open another terminal window (keep backend running).

Navigate to the frontend folder:
cd frontend
Install dependencies:
npm install
Run the frontend:
npm run dev
